------------------------------------------------------------------------
DESCRIPTION
------------------------------------------------------------------------
This project is a responsive Personal Portfolio Website designed using 
Semantic HTML5 and CSS3. The goal is to showcase technical skills, 
academic projects, and personal interests in a modern, visually appealing 
layout.

------------------------------------------------------------------------
FOLDER STRUCTURE
------------------------------------------------------------------------
portfolio/
│
├── index.html        # Main HTML file containing the website structure (5 Sections)
├── style.css         # CSS file for styling, layout, and color themes
├── README.txt        # Project documentation (this file)
│
└── images/           # Folder containing image assets
    └── profile.jpg   # Profile picture used in the Home Section

------------------------------------------------------------------------
SECTIONS INCLUDED
------------------------------------------------------------------------
1. Home: Introduction with a photo and tagline.
2. About: Brief bio highlighting CSE (AI & ML) background.
3. Skills: Technical skills display.
4. Projects: Overview of 3 academic projects:
    - Tourism Website
    - Virtual Art Gallery
    - HealthHack terf
5. Contact: Contact information and a message form.

------------------------------------------------------------------------
TOOLS USED
------------------------------------------------------------------------
- HTML5 (Structure)
- CSS3 (Styling, Grid, Flexbox)
- VS Code (Editor)
